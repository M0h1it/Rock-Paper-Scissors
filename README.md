# ✊✋✌️ Rock Paper Scissors

A fun and interactive **Rock Paper Scissors** game built using **HTML**, **CSS**, and **JavaScript**. Challenge the computer, track your score, and enjoy a simple yet engaging browser-based gaming experience.

> **Live Demo:** https://m0h1it.github.io/Rock-Paper-Scissors/ *(Update if your deployment URL is different.)*

---

## 📖 Overview

This project recreates the classic **Rock Paper Scissors** game with an intuitive user interface and real-time gameplay. It demonstrates fundamental JavaScript concepts such as event handling, DOM manipulation, randomization, conditional logic, and score management.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/M0h1it/Rock-Paper-Scissors.git
```

### Navigate to the project

```bash
cd Rock-Paper-Scissors
```

### Run the project

Simply open the `index.html` file in your preferred web browser.

For a better development experience, use **VS Code Live Server**.

---

## 🎮 How to Play

1. Click on **Rock**, **Paper**, or **Scissors**.
2. The computer randomly selects its move.
3. The winner is determined based on the classic rules:

   * 🪨 Rock beats Scissors
   * ✂️ Scissors beats Paper
   * 📄 Paper beats Rock
4. The score updates automatically after every round.
5. Continue playing as many rounds as you like.
